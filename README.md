## ğŸš¦ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd ai-shopping-app
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
